# Ambient Audio

Place your ambient background audio file here.

**Supported formats:** WAV, MP3, M4A, OGG

**Example:**
- `ambience.wav` - Continuous background atmosphere

**Tips:**
- Use loopable ambient tracks (seamless loops work best)
- Keep volume low (0.10 to 0.20 range in config.json)
- Should be subtle and not distract from narration
- Longer tracks are better (2-5 minutes)
- The script will automatically loop the ambience to match video duration

**Good ambience types for horror:**
- Rain and thunder
- Wind howling
- Eerie drones
- Forest at night
- Distant rumbling
- Creaky building sounds
- Subtle music/pads

**Free ambience sources:**
- Freesound.org
- YouTube Audio Library
- MyNoise.net (generate custom ambiences)
- Ambient-Mixer.com
