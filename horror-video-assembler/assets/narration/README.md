# Narration Audio

Place your main narration audio file here.

**Supported formats:** WAV, MP3, M4A, OGG

**Example:**
- `narration.wav` - Your main voiceover from ElevenLabs or other TTS service

**Tips:**
- Use high-quality audio (44.1kHz or 48kHz sample rate)
- Normalize audio levels before exporting
- Consider adding slight compression for consistent volume
